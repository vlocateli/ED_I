# ED_I

**LINUX**

To compile:
 ```bash
 g++ -Wall -Wextra -Werror <file_name>.cpp -o prg
 ```
To execute:
 ```bash
 ./prg
 ```
**Windows**
To compile (using **MinGW** or **WSL** with g++ installed:
```powershell
g++ -Wall -Wextra -Werror <file_name>.cpp -o prg
```

To execute:
 ```powershell
 ./prg.exe
 ```
